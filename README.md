# BeepeeM is a BPM tapping application that is both simple and lightweight

## Installation

To get started, clone the repo:

```bash
git clone https://codeberg.org/dalentri/beepeeM.git
cd beepeeM
```

## Running the Application

Then, to run the application:

```bash
python3 src/main.py
```

Now tap along to music and enjoy the live BPM reading in the terminal.
